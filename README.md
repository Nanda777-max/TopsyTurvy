# TopsyTurvy — UI Recreation Challenge

A college competition project where participants were given the UI of an existing website and had to recreate it from scratch using only HTML and CSS.

## About
Given a reference design (IEEE chapter website), the goal was to pixel-perfect recreate the UI — no frameworks, just raw HTML and CSS. Built under competition conditions.

## Tech Stack
- HTML
- CSS

## Structure
```
ieee/
  index.html    # Main page
  styles.css    # All styling
  i/            # Images used in the site
```

## Context
Submitted as part of a UI recreation competition held at college. The challenge tested attention to detail, layout skills and CSS proficiency.
